# RoutesBus - Starokostiantyniv

Interactive bus map for Starokostiantyniv, Khmelnytskyi Oblast.
Built with **Nuxt 4** and **Leaflet**.

## Features

- **Interactive Map**: Real-time visualization of bus routes and stops using Leaflet.
- **Route Tracking**: View routes 1 (Bus St - Railway St) and 5 (Hospital - Castle).
- **Arrival Estimates**: Click on any stop to see simulated upcoming arrival times.
- **Modern UI**: Glassmorphism design with responsive layouts.

## Setup

```bash
# Install dependencies
npm install

# Start the development server
npm run dev
```

## Tech Stack

- **Framework**: Nuxt 4 (Latest)
- **Map**: Leaflet + @vue-leaflet/vue-leaflet
- **Styling**: Vanilla CSS (Scoped & Global)
- **Data**: Mock data simulating local transport schedule
